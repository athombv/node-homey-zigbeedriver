# homey-zigbeedriver

## Introduction
This module is used to make the creation of ZigBee apps easier.

It is essentially a map-tool from Homey-capabilities to ZigBee endpoints and clusters.

## Installation

```bash
$ npm install homey-zigbeedriver
```
